Smart Crop Selector is an intelligent web-based platform designed to assist farmers, agricultural researchers, and enthusiasts in making data-driven decisions about crop selection, seed recommendations, and crop-specific information. Using a Machine Learning model, the system provides accurate crop predictions based on Temperature, pH Value etc.

Features Overview

    Input Parameters:

  • Nitrogen (N): Enter nitrogen content in the soil (mg/kg).
  • Phosphorus (P): Enter phosphorus content in the soil (mg/kg).
  • Potassium (K): Enter potassium content in the soil (mg/kg).

  • Humidity: Enter the humidity percentage (%).
  • pH Value: Enter the soil pH level.
  • Rainfall: Enter the annual rainfall (mm).

    • Output: The system analyzes the input data using a Machine Learning model and suggests the most suitable crop for cultivation.

Seed Recommendation System

    The Seed Recommendation System suggests the best-quality seeds for a given crop based on store jsonBIN data.

    Input Parameters:

    Enter the name of the crop (e.g., "Wheat", "Rice", "Maize").

    Output: A list of recommended high-yielding, disease-resistant seed varieties suitable for the selected crop.
